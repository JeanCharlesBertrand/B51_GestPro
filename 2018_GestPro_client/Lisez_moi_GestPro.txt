GestPro version 0.1

Bienvenue à la gestion de projet en services à la carte.

GestPro est un ensemble d'applications servant à gérer les divers aspects du développement logiciel, suivant la méthode agile.

Le logiciel se compose d'un serveur fournissant à la demande les modules, les applications spécialisées,
permettant de gérer les étapes du développement de projet, de l'analyse du mandat jusqu'au rapport de livraison finale.


Le Serveur GestPro

Il faut d'abord installer le serveur, et laisser connaitre sn adresse IP et le port sur lequel se fera le service.


Le client GestPro permet de se connecter à l'acdresse IP du serveur en fournissant le nom de l'usager.

Le client peut maintenant demander un module d'application spécifique.
